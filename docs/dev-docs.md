# Collie WebApp 功能说明

## 1. Bugreport Hub
- 上传 bugreport（.txt / .zip，支持拖拽）
- 场景分析：动态性能、九大场景驻留、被杀定点、自选应用
- 进度与状态：上传/分析/待确认/完成/失败
- 结果产出：HTML 可视化、文本报告、设备信息、关键片段
- 内存与事件：内存时间线、事件详情、重点进程视图
- 智能能力：单报告 AI 解读、双报告 AI 对比
- 结果处理：在线预览、下载、打包下载
- 任务管理：任务列表、删除、对比模式、同场景限制

## 2. Utilities Hub（实用工具）
- 设备连接：设备列表、刷新、共用设备条
- 应用安装：基于预设或自定义包名安装
- 版本检查：历史记录、差异对比、Markdown 输出
- 设备内存采集：采集 meminfo / vmstat 等基础数据
- 内存设计对比：对比设备内存设计参数
- 编译与预启动：编译进度与日志跟踪
- Meminfo 摘要：生成结构化 meminfo 摘要与原始查看

## 3. R&D Hub（研发自测）
- 存活监控：轮询目标进程，记录首次被杀与上下文
- 存活检查：按包名快速检查存活与 meminfo
- 连续启动-驻留：Collectors/PreStart/Bugreport 组合流程
- Simpleperf 抓取：性能采样与火焰图预览

## 4. 体验与展示
- 主题切换：星河蓝 / Shell / 我的世界
- 首页引导：首次使用 3 步指引（可关闭）
- 最近 7 天趋势：任务数量、完成数、Top 场景
- 动效与动线：模块入场、卡片悬浮、进度反馈

## 5. 输出与协作
- 多格式导出：HTML / Markdown / TXT / ZIP
- 飞书导出：对比结果一键保存到飞书文档
